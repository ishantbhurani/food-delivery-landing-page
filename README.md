## Run Locally

Clone the project

```bash
  git clone git@github.com:ishantbhurani/food-delivery-landing-page.git
```

Go to the project directory

```bash
  cd food-delivery-landing-page
```

Run `index.html`

```bash
  <browsername> index.html
```

E.g.

```bash
  firefox index.html
```

```bash
  google-chrome index.html
```
